What is Remote 3D?

This project tackles a major issue for current 3D rendering tehnologies in the browser: limited CPU and GPU resources. Our solution is suited for mobile and low powered devices or embedded graphical systems with limited processing power or for old systems incompatible with the new WegGL technology.
Generate 3d objects in order to better visualise them
Select visible objects or faces with a mouse click
Select from a variety of objects to be rendered
Generate complex custom objects from text input
Move the frame and see the object from different angles
The generated object moves in real time without using your computer power
